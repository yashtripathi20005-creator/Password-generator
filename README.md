# Password Generator Pro

A comprehensive password generator with both GUI and CLI interfaces, built with Python.

## Features

- **Secure Password Generation**: Generate strong, random passwords
- **Customizable Options**: 
  - Adjustable length (4-64 characters)
  - Include/exclude uppercase, lowercase, digits, symbols
  - Exclude ambiguous characters (il1Lo0O)
  - Minimum characters from each type
- **Memorable Passphrases**: Generate easy-to-remember but secure passphrases
- **Password Strength Evaluation**: Get detailed analysis of password strength
- **History Management**: Save and view previously generated passwords
- **Clipboard Support**: Copy passwords with one click
- **Dual Interface**: Both GUI and command-line versions available
